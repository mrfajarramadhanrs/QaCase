# Automation Strategy

## Scope
Automation focuses on validating critical and frequently executed user journeys to ensure confidence during regression cycles.

## Included Scenarios
- Authentication happy paths
- Core functional flows
- Data submission with valid inputs

## Excluded Scenarios
- Visual/UI verification
- One-time edge cases
- Scenarios dependent on external services

## Tooling
Playwright is selected due to:
- Cross-browser support
- Reliable handling of modern web applications
- Strong debugging capabilities

Appium is selected due to:
- Able to automated Android and iOS
- Availbility for all programming language
- Stable
